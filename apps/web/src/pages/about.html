About Toolpad

A free, open-source suite of developer utility tools — built for the developer community.

What is Toolpad?

Toolpad is a free collection of browser-based developer utility tools. Every tool runs in your browser — no account needed, no data stored, no limits. A free REST API is also available.

Our tools

  • Data generators — Fake names, addresses, emails, phones, credit cards (Luhn-valid), IBANs (MOD-97), IMEI, MAC, IP, UUID. All structurally correct.
  • Formatters — Prettify, minify, and validate JSON and XML with clear error messages.
  • Converters — JSON↔XML, JSON↔CSV, Base64, Hex, Text→JSON/XML, DOCX→PDF.

Why we built this

Every developer needs test data. We wanted one clean, fast, no-friction place — no accounts, no row limits, with a REST API for programmatic access.

Technical details

  • Frontend: Plain HTML, CSS, vanilla JavaScript — no frameworks
  • Backend API: Node.js + Express
  • All browser tools: 100% client-side, no server calls
  • Hosting: Vercel global CDN
