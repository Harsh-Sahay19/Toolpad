API documentation

The Toolpad REST API lets you access all generators and data tools programmatically. No API key required. Base URL: https://api.toolpad.in/api/v1

Endpoints
Personal Financial Network & Device Location Data tools Combined

Response format

All endpoints return JSON with a standard envelope:

{
  "success": true,
  "data": { ... },
  "generatedAt": "2026-03-31T06:00:00.000Z"
}

On error:

{ "success": false, "error": "Error description" }
