About this DOCX to PDF converter

This tool converts Microsoft Word documents (.docx format) to PDF. It preserves text formatting, headings, paragraphs, and basic styling. Complex documents with embedded images, charts, or tracked changes may have some formatting differences in the output.

What is supported

  • Standard text formatting — bold, italic, underline, font sizes
  • Headings (H1 through H6)
  • Paragraphs, line breaks, and basic layout
  • Tables with simple cell content
  • Lists (bulleted and numbered)

API access

Convert programmatically: POST /api/v1/doc/docx-to-pdf with the .docx file as multipart form data.

Is my document kept private? +
Files are processed server-side for conversion (PDF generation requires server processing), but are deleted immediately after conversion. No document content is stored or logged.
What is the maximum file size? +
Files up to 10MB are supported. For larger documents, consider splitting the document or using the API endpoint which also supports up to 10MB per request.
Does it support .doc (older Word format)? +
Currently only .docx (Office Open XML) format is supported. To convert .doc files, open them in Microsoft Word or LibreOffice and save as .docx first.
